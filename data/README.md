# Data

Milestone 1 uses a small generated sample dataset so the complete pipeline can run without downloading large files.

Final project dataset options:

1. PlantVillage dataset for crop disease classification.
2. ICAR crop management bulletins for RAG-based agricultural advisories.
3. Region-specific public agriculture advisories for localization.

Do not commit large datasets to GitHub. Keep them inside `data/raw/` locally.
